  • StoreProtocol

    Defines requirements of Stores. Default implementation of this protocol is the Store class. Application should only use one store for the entirety of applicaction state. Stores dispatch actions to their defined reducer, which in turn return the new value of state. With the help of ObservableObject Store notifies it’s subscribers of state changes.
